## ğŸš€ Quick Start

### Prerequisites
- **Node.js 18+** - JavaScript runtime
- **npm or bun** - Package manager
- **Git** - Version control

### Installation

```bash
# Clone the repository
git clone https://github.com/Aqib2607/portfolio.git
cd portfolio

# Install dependencies
npm install
# OR using bun (faster)
bun install

# Set up environment variables
cp .env.example .env
# Edit .env with your configuration

# Start development server
npm run dev
# OR using bun
bun run dev
```

### Build for Production

```bash
# Build the project
npm run build
# OR using bun
bun run build

# Preview the build
npm run preview
# OR using bun
bun run preview
```
